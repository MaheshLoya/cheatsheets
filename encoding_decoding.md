# Cheatsheet: Encoding & Decoding
### Python
#### char to hex
``` ‘l’.encode(‘hex’) == ‘6c’ ```

#### hex to char 
``` '6c'.decode('hex') == ‘l’ ```

#### char to int
``` ord(‘l’) == 108 ```

#### int to char
``` chr(108) == ‘l’ ```

#### hex to int
``` int('21',16) ```
